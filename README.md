**#hash**
**Password Checker (Bcrypt)**
A simple CLI tool to hash and verify passwords.

**Installations**
*npm install* 
*bcrypt*

**Usage**
Run the script with a password as an argument:
*node hash.js your_password*

If password.txt is empty: The script hashes and saves your password.

If a hash exists: The script compares your input and returns +correct data or -incorrect data.
